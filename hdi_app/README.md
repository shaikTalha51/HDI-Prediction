# HDI Predictor — Flask Web App

## Setup
```
pip install numpy pandas matplotlib seaborn scikit-learn flask
```

## Run order
1. `python generate_dataset.py` -> creates hdi_dataset.csv
   (swap this for a real Kaggle HDI dataset download in your own environment)
2. `python train_model.py` -> runs EDA, trains the model, saves
   hdi_model.pkl / scaler.pkl / metrics.pkl and the EDA charts
3. `python app.py` -> starts the Flask server at http://127.0.0.1:5000

## Pages
- `/`         Home — project overview, tier legend, model metrics
- `/predict`  Input form for the 4 indicators -> prediction result
- `/insights` EDA charts (heatmap, scatter, strip plot, distribution)

## Files
- generate_dataset.py   synthetic HDI dataset generator (Epic 3)
- train_model.py        EDA + preprocessing + train/test split + Linear
                         Regression + pickle save (Epics 2-7)
- app.py                 Flask routes (Epic 8)
- templates/             base.html, index.html, predict.html, result.html, insights.html
- static/style.css       app styling
- static/eda_*.png       charts generated by train_model.py

## Swapping in a real dataset
Replace generate_dataset.py's output with a real HDI CSV (e.g. from Kaggle)
that has these columns, then re-run train_model.py:
  Country, Life_Expectancy, Mean_Years_Schooling,
  Expected_Years_Schooling, GNI_Per_Capita, HDI_Score
